## ⚡️ Быстрый старт (CLI)

### Шаг 1. Установка GitHub CLI
Установите утилиту `gh` через пакетный менеджер вашей операционной системы:

| Операционная система | Команда для установки |
| :--- | :--- |
| **Windows** | `winget install --id GitHub.cli` |
| **macOS** | `brew install gh` |

### Шаг 2. Авторизация в GitHub
Выполните команду в терминале и следуйте инструкциям на экране для привязки аккаунта:
```bash
gh auth login
```

### Шаг 3. Активация скрипта
Запустите команду активации локального Dart-пакета:

| Операционная система | Команда |
| :--- | :--- |
| **Windows** | `dart pub global activate --source git git@github.com:linarchingizov1-ux/MUZHIKI_CLI.git` |
| **macOS** | `echo 'export PATH="$PATH:$HOME/.pub-cache/bin"' >> ~/.zshrc source ~/.zshrc` |

> 💡 **Примечание для macOS:** После выполнения команды обязательно перезапустите терминал, чтобы применились переменные окружения.

## 🔄 Обновление скрипта

Если вы внесли изменения в исходный код скрипта, его необходимо пересобрать. Для этого деактивируйте текущую версию и активируйте её заново.

### Полезные команды

* **Посмотреть список активных пакетов (узнать, из какого проекта запущен):**
  ```bash
  dart pub global list
  ```
* **Деактивировать старую версию (деактивируй все что покажет из мужики чтобы пакет работал с репозитория):**
  ```bash
  dart pub global deactivate muzhiki_cli
  ```
* **Активировать обновленную версию из корня репозитория:**
  ```bash
dart pub global activate \
  --source git \
  git@github.com:linarchingizov1-ux/MUZHIKI_CLI.git
  ```
